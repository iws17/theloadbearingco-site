# The Load Bearing Company

Public teaser source for [theloadbearingco.com](http://theloadbearingco.com/).

## Public boundary

This repository is intentionally limited to the public-facing teaser. It does
not disclose products, partners, suppliers, pricing, timing, methods, team,
roadmap, or launch mechanics.

The page is a six-section visual statement built from static HTML, CSS,
self-hosted type, and anonymous artwork. It collects no information and makes
no third-party requests: no forms, tracking, analytics, cookies, or APIs.

## Files

- `index.html` — the teaser
- `404.html` — the branded not-found page
- `assets/` — self-hosted typography and artwork used by the public page

The operational details behind the site live outside this public repository.
